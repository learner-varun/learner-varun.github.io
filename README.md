## 🚀 Performance-First Portfolio Design

This portfolio is built to demonstrate engineering best practices not just in QA, but in **web performance and clean code architecture**. It has been optimized for speed, battery conservation, and lightweight page weight:

1.  **Rendering Optimization (O(N²) to O(1) checks):** Optimized the custom background particle connection logic. By checking squared distances first, we skip expensive CPU square root calculations (`Math.sqrt`) for 90% of particle pairs.
2.  **Dynamic Device Scaling:** The particle simulation automatically scales its density based on viewport dimensions (from 70 particles on Desktop to 15 on Mobile), reducing overhead on low-end processors.
3.  **Zero Scroll Lag (Jank-Free UX):** Replaced heavy, main-thread window `scroll` event listeners with modern, browser-optimized `IntersectionObserver` handlers to toggle active menu links.
4.  **Deferred Non-Blocking Loading:**
    *   `Chart.js` and custom scripts are executed using `defer` to prevent blocking the initial document parsing.
    *   Google Fonts are preconnected to reduce DNS lookups.
    *   FontAwesome is loaded asynchronously via print media-query techniques so that the page paints immediately.
5.  **Hardware Accelerated Animations:** Added CSS `will-change: transform` hints on complex glassmorphic elements containing backdrop blur filters, forcing the browser to promote them to GPU compositor layers and ensuring smooth 60 FPS scrolling.
